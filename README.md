-- High-z Galaxy Formation and Dust Attenuation Models

This repository contains a set of Python scripts used to model the formation, dust content, and radiative properties of high-redshift galaxies, focusing on how UV and IR luminosities are affected by ISM geometry, dust microphysics, and turbulence.

Current paper draft: https://www.overleaf.com/read/rsnzrzvdkyvm#bbf2a2

-- Repository Structure

dust_JWST_z10_GSD.py
Computes dust microphysical quantities (absorption, scattering, extinction) starting from a grain size distribution including silicates, carbonaceous grains, and PAHs. All opacity functions used elsewhere in the repository come from this module.

highz_gal_SAM.py
Core semi-analytic model. Populates the halo mass function with galaxies and computes their time-resolved star-formation histories and dust buildup using SN dust yields (no dust ejection assumed).

dust_JWST_z10_exploring_params_last.py
Computes individual-galaxy properties at fixed redshift (typically z ≳ 7–10): stellar mass, SFR, optical depth, and intrinsic and dust-attenuated UV magnitudes. Assumes a uniform ISM distribution.

dust_JWST_z10_clumpy_ISM.py
Same as the above script but including the effect of clumpy ISM geometry, modified grain size distribution, and sightline-to-sightline attenuation arising from a log-normal Σ_d distribution generated by ISM turbulence (parameterized by the Mach number).

dust_JWST_z10_population_LF_last.py
Older version of the population-level UV and IR luminosity function computation. Includes halo spin-parameter scatter and explores grids in (ε_star, y_d). Kept for reference but superseded by the sampling-based code below.

dust_JWST_z10_population_LF_draw_sample_uniform.py
Most up-to-date module for the uniform ISM case. Uses Monte Carlo sampling of many lines of sight per halo to compute attenuation and the corresponding UV and IR luminosity functions. Preferred over dust_JWST_z10_population_LF_last.py.

dust_JWST_z10_population_LF_turbulent.py
Dedicated to the turbulent ISM case. Contains two versions of the LF computation:
(1) a Monte Carlo sightline-sampling approach that draws Σ_d values from a log-normal distribution determined by the Mach number;
(2) the new cumulative-distribution method (used in the paper), where the cumulative UV LF is obtained by integrating over the Σ_d distribution implied by the Mach-number PDF, and the differential UV LF is then derived from this mapping. This method is faster, more stable, and currently adopted in the paper.

-- Modeling Logic

highz_gal_SAM.py → populate halos with galaxies
dust_JWST_z10_exploring_params_last.py → individual-galaxy properties (uniform ISM)
dust_JWST_z10_clumpy_ISM.py → individual-galaxy properties including clumpy/turbulent ISM effects
dust_JWST_z10_population_LF_draw_sample_uniform.py → UV and IR luminosity functions for uniform ISM
dust_JWST_z10_population_LF_turbulent.py → UV and IR luminosity functions including turbulence and Mach-number distributions

-- Physical Treatment Summary

Star formation and dust buildup:
Computed self-consistently from the SAM. Dust production is from SN; no dust outflows are assumed.

UV emission:
Computed along multiple random sightlines for each halo. Attenuation depends on ISM geometry and dust distribution. In clumpy/turbulent ISM versions, Σ_d along each line of sight is drawn from a log-normal distribution set by the Mach number, capturing geometric and turbulence-driven scatter.

IR emission:
Assumed isotropic. Computed via energy conservation by integrating the absorbed UV fraction over the full Σ_d distribution.

Luminosity functions:
Two approaches exist:
(1) sampling-based LFs from Monte Carlo sightline attenuation (used in uniform and turbulent cases);
(2) cumulative-distribution LFs (turbulent fiducial), where the distribution of Σ_d from the Mach-number PDF is integrated to compute the cumulative UV LF and then inverted to obtain the final UV LF. This is the method used in the current paper.
